Morse Code Translator.

This project takes any inserted English language word, phrase, or sentence, and converts it into morse code.
All the same, it can also take a sentence in morse code and translate it to English.
=================================================================================================================
Example:
Input: Hello World
Output: .... . .-.. .-.. ---   .-- --- .-. .-.. -..

OR

Input: .... . .-.. .-.. ---   .-- --- .-. .-.. -..
Output: hello world
=================================================================================================================
Project status:
Mostly operational. Base functions are firmly in place. The only issue is that the English translator
lacks spaces between its words when translated from morse code.
=================================================================================================================
Possibility for future updates:
